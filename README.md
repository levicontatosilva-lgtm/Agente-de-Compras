# 📊 Agente de Análise de Compras

Este aplicativo foi desenvolvido para automatizar a análise de consumo e geração de listas de compras a partir de planilhas de estoque.

## 🚀 Como colocar o aplicativo no ar (Grátis)

Siga estes passos simples para hospedar seu aplicativo gratuitamente no **Streamlit Cloud**:

1.  **Crie uma conta no GitHub**: Se ainda não tiver, crie em [github.com](https://github.com).
2.  **Crie um novo Repositório**:
    *   Dê um nome como `meu-agente-compras`.
    *   Deixe como **Público** (ou Privado, se preferir).
3.  **Suba os arquivos**:
    *   Suba os arquivos `app.py`, `requirements.txt` e a pasta `.streamlit` para o seu repositório.
4.  **Conecte ao Streamlit Cloud**:
    *   Acesse [share.streamlit.io](https://share.streamlit.io).
    *   Faça login com sua conta do GitHub.
    *   Clique em **"New app"**.
    *   Selecione o repositório que você criou, a branch `main` e o arquivo `app.py`.
    *   Clique em **"Deploy!"**.

Em poucos minutos, seu aplicativo estará online em um link que você poderá acessar de qualquer lugar!

## 🛠️ Funcionalidades

*   **Upload de Planilha**: Compatível com o formato de exportação de consumo (13 dias).
*   **Margem de Segurança**: Ajuste global ou específico por categoria.
*   **Lote Mínimo**: Configure quantidades mínimas de compra para evitar pedidos fracionados.
*   **Exportação**: Gere uma nova planilha Excel com a lista de compras pronta.
